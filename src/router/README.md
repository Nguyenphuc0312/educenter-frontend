# Router

Project hien chua cai `vue-router`; menu dang switch component truc tiep trong `src/App.vue`.
Giai doan demo CRUD co the giu cach nay de tranh them dependency.

Khi can URL rieng cho tung trang, cai `vue-router` va chuyen menu trong `App.vue` sang router-link.

Route du kien:

- `/`
- `/courses`
- `/schedules`
- `/students`
- `/enrollments`
- `/attendances`
- `/results`
- `/payments`
- `/reports`
